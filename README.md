# Dr. Steve Brule Name Generator

I was curious on how PyPi packages were created, so this is what I came up with for testing purposes.
It's pretty handy for making a lists of names for example data, etc.

### Installation
---

```
C:\> pip install brule
```
### Usage
---
```python
>>>import brule as b
>>>b.randomName()
'Cindy Brangle'
```
### Help
---
```python
>>>import brule as b
>>>b.help()
```
